Este es un artículo de ejemplo.

Puedes crear más archivos dentro de la carpeta `articles/` usando extensión `.md`.

Frontmatter soportado:

- `title`
- `slug`
- `excerpt`
- `publishedAt`
- `draft`

Si omites `slug`, se usa el nombre del archivo.
